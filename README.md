#  Dar Zitoun Twin API

API RESTful de gestion d’un restaurant marocain.  
Développée avec **Node.js**, **Express**, **TypeScript**, **MongoDB** (Mongoose), et sécurisée avec **JWT**.

##   Fonctionnalités

- Gestion des plats (CRUD, filtrage par catégorie, prix, disponibilité)
- Gestion des chefs (spécialités, expérience)
- Authentification JWT (rôles : client, livreur, admin)
- Commandes : calcul automatique du montant, frais de retard à la livraison
- Validation des données avec Zod
- Documentation interactive Swagger UI
- Architecture MVC (modèles, services, contrôleurs, middlewares)
- Tests unitaires (Jest / Supertest)
- Sécurité : Helmet (CSP personnalisé), CORS, bcrypt
